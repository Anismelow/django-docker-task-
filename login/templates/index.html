Bienbenido a Django Task

Este es un proyecto de una página de tareas que incorpora funciones básicas de un CRUD, así como un sistema de inicio de sesión para acceder a la página y un registro de usuario que solo muestra las tareas creadas por el usuario.

Este proyecto ha sido desarrollado para poner en práctica mis conocimientos sobre Django y las diversas bibliotecas que utiliza para el manejo de usuarios y bases de datos, incluyendo PostgreSQL. Al finalizar, crearé una imagen de Docker y, utilizando Docker Compose, desplegaré tanto la imagen como la base de datos en contenedores. Finalmente, desplegaré la aplicación en un clúster de ECS de AWS.
